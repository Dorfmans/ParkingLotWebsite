## ParkingLotWebsite

  #### Parking Lot (API)
  #### Parking Lot APP (Website)

### Snipets to install modules and run on localhost


#### Parking Lot:

  ##### npm init

  ##### npm run dev

#### Parking Lot App:

  ##### npm init
